Single Sign On (SSO) into WordPress (WP) using YourMembership credentials - Login with YourMembership [24/7 SUPPORT]

== Description ==
<a href="https://plugins.miniorange.com/wordpress-yourmembership-single-sign-on-sso">YourMembership Single Sign On (SSO)</a> for WordPress allows SSO login with YourMembership (YM) credentials into your WordPress site.

<a href="https://plugins.miniorange.com/wordpress-yourmembership-single-sign-on-sso">YourMembership Single Sign On (SSO)</a> provides authentication for WordPress allowing your users to login to the WP site.

= Free Version Features =
*   YourMembership Single Sign On supports SSO with YourMembership (YM).
*   **New API Support** - Configure your plugin using the NEW REST APIs.
*   **Unlimited Authentications** – Unlimited authentications with YourMembership (YM).
*   **Auto Create Users** - Users will be auto-created in WordPress after SSO from YourMembership (YM).
*   **Login Widgets** - Use YourMembership (YM) Login Widgets to easily integrate the YourMembership (YM) SSO link with your Wordpress site.
*   **Attribute Mapping** - The NameID value sent by YourMembership (YM) will be mapped to the Username and Email of your WordPress users.
*   **Step-by-step Guides** - Use step-by-step guide to configure YourMembership (YM).

= Premium Version Features =
*   Includes all YourMembership **FREE** version features.   
*   **Redirect to YM Login Page for Authentication** - Users will be redirected to the YourMembership (YM) login page for authentication.
*   **Login Widgets and Short Code** - Use Widgets to easily integrate the YourMembership (YM) link with your Wordpress site. Use Short Code (PHP or HTML) generated by YourMembership Single Sign On to place the YourMembership login link wherever you want on the site.
*   **Attribute Mapping** - YourMembership Single Sign On provides the feature to map your YourMembership (YM) profile attributes to your WordPress site.
*   **Auto-redirect to YourMembership** - Users will be redirected to YourMembership (YM) for SSO when trying to access the WordPress login page.
*   **Protect Site** - Users trying to access Wordpress will be redirected to YourMembership (YM) login page for SSO.

= All-inclusive Version Features =
*   Includes all YourMembership **PREMIUM** version features.
*   **Single Logout** - Support for YourMembership (YM) Single Logout (Works only if YM supports SLO).
*   **Advanced Attribute Mapping** - YourMembership Single Sign On provides the feature to map your YourMembership (YM) profile attributes to your WordPress site attributes like Username, Email, First Name, Last Name, Group/Role, Display Name.
*   **Advanced Role Mapping** - YourMembership Single Sign On provides the feature to assign WordPress roles to your users based on the group/role profile attribute sent by YourMembership (YM).
*   All YourMembership (YM) add-ons included like Page Restriction, BuddyPress Integrator, LearnDash Integrator, etc.
*   **WordPress Multi-site Support** - Multi-Site environment is one which allows multiple subdomains / subdirectories to share a single installation. With multisite YourMembership (YM) premium plugin, you can configure the YourMembership (YM) in minutes for all your sites in a network. While, if you have basic YourMembership premium plugin, you have to do plugin configuration on each site individually.

    For Example - If you have 1 main site with 3 subsites. Then, you have to configure the plugin 3 times on each site. Instead, with multisite YourMembership premium plugin, you have to configure the plugin only once on main network site.

= Add-ons =
We have a variety of add-ons that can be integrated with the YourMembership SSO plugin to enhance the YourMembership SSO functionality of your WordPress site.

*	**<a href="">Page Restriction</a>** -  	This add-on is basically used to protect the pages/posts of your site with YourMembership (YM) login page and also, restrict the access to pages/posts of the site based on the user roles.
*	**<a href="">BuddyPress Integration</a>** - This add-on maps the attributes fetched from YourMembership (YM) with BuddyPress attributes.
*	**<a href="">LearnDash Integration</a>** - This add-on will map the YourMembership SSO user to LearnDash groups as per the profile attributes sent by YourMembership (YM).
*	**<a href="">SSO Login Audit</a>** -  	SSO Login Audit captures all the YourMembership SSO users and generates detailed session reports.
*	**<a href="">Attribute Based Redirection</a>** - ABR add-on helps you to redirect your users to different pages after they authenticate via YourMembership SSO into your site, based on the attributes sent by YourMembership (YM).
*	**<a href="">Media Restriction Add-On</a>** - Restrict any kind of media files such as images, audio, videos, documents, etc.
*	**<a href="">SSO Session Management</a>** - SSO session management add-on manages the login session time of your users based on their WordPress roles.

== Installation ==

= From your WordPress dashboard =
1. Visit `Plugins > Add New`.
2. Search for `YourMembership Single Sign On`. Find and Install `YourMembership Single Sign On – YM SSO login`.
3. Activate the plugin from your Plugins page.

= From WordPress.org =
1. Download YourMembership Single Sign On – YM SSO login plugin.
2. Unzip and upload the `login-with-yourmembership` directory to your `/wp-content/plugins/` directory.
3. Activate YourMembership SSO from your Plugins page.
